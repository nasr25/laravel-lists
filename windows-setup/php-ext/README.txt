PHP OCI8 EXTENSION FOR WINDOWS
============================================

BEFORE going offline, download the PHP OCI8 extension:

URL: https://windows.php.net/downloads/pecl/releases/oci8/

Download the version matching your PHP:
- For PHP 8.3 Thread Safe x64: php_oci8-3.4.1-8.3-ts-vs16-x64.zip
- For PHP 8.2 Thread Safe x64: php_oci8-3.4.1-8.2-ts-vs16-x64.zip

Extract and copy php_oci8.dll to your PHP ext folder:
C:\php\ext\php_oci8.dll

Then enable in php.ini:
extension=oci8
